# 高校宿舍报修管理系统运行说明

## 1. 环境要求

- JDK 17
- Maven 3.9 及以上
- Node.js 18 及以上
- Windows PowerShell

## 2. 推荐启动方式

在项目根目录执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\start-dev.ps1
```

启动后访问：

- 统一门户首页：`http://localhost:5183/home`
- 统一登录页：`http://localhost:5183/login`
- 后端接口：`http://localhost:2360`

## 3. 手动启动方式

### 3.1 启动后端

```powershell
cd server
cmd /c "mvn -Dspring-boot.run.arguments=--server.port=2360 spring-boot:run"
```

### 3.2 启动统一前端

```powershell
cd admin-web
npm install
$env:VITE_API_BASE_URL='http://localhost:2360/api'
npm run dev
```

- 维修人员 2：`repair02 / 123456`

- 学生 2：`student02 / 123456`

## 5. 说明

- 项目使用 SQLite，首次启动后端时会在根目录自动生成 `dorm_repair.db`
- 首次启动统一前端前建议执行一次 `npm install`
- 如果 `2360` 或 `5183` 端口被占用，需要先释放对应端口
- 如需重新生成初始化数据，可删除 `dorm_repair.db` 后重新启动后端
